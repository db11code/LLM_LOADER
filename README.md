# LLM Loader

_Initial project scaffold generated during Milestone 1._

## Getting Started (Dev)

```bash
git clone <repo>
cd LLM_LOADER

# Python backend
python -m venv .venv
.venv\Scripts\activate
pip install -e .

# Front-end
cd frontend
npm install          # postinstall auto-builds preload.cjs
npm run dev          # launches Electron + Vite
```
